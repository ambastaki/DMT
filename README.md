# DMT Engineering Firm Assessment Helper

This repository contains a lightweight web tool to help users assess and evaluate engineering firms while referencing the official DMT engineering firms inquiry page:

https://pages.dmt.gov.ae/en/engineering-firms-inquiry

## Usage

1. Open `/home/runner/work/DMT/DMT/index.html` in a browser.
2. Enter the firm name and optional inquiry page link.
3. Set each criterion score from 1 (low) to 5 (high).
4. Click **Calculate Assessment** to get:
   - weighted overall score (out of 100),
   - performance category,
   - recommendation text.
